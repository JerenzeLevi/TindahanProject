# 📸 Product Inventory Images

This folder is the storage hub for all product images uploaded via the **Admin Portal**.

### ⚠️ Important Note
* **Dynamic Content:** The files in this folder are uploaded manually by the store administrator (My Tita). Don't ya dare think of any malicious -.-
* **Naming Convention:** Images are automatically prefixed with a Unix timestamp (e.g., `1715580000_coke.jpg`) to prevent file overwriting.
