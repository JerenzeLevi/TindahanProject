# 🎨 The Aero Engine (Stylesheets)

This folder contains the core visual logic for **Guillermo & Lucina's Tindahan**. The styles are designed to emulate the **Frutiger Aero** aesthetic (2004-2013).

### 📂 File Map
* **`admin.css`** 🛠️: Styles for the Command Center. Includes specific logic for the glass-card inventory system and dark-blue editable inputs.
* **`store.php`** 🛒: Styles for the public storefront, featuring the "Digital Shelves" grid and glossy category buttons.
* **`style.css`** 🌊: The main landing page styles, including the sky-to-grass gradients and cursive "Tindahan" header.
* **`about.css`** 🌿: Styles for the conect and a little history of our Tindahan.

### ✨ Visual Principles Used:
1.  **Glassmorphism:** High `backdrop-filter: blur()` values.
2.  **Skeuomorphism:** Glossy overlays on buttons and cards.
3.  **Vibrant Gradients:** Linear flows from `#ffffff` to `#39ff14`.
4.  **Pill Shapes:** Extreme `border-radius` (25px to 50px) for a "bubbly" feel.
